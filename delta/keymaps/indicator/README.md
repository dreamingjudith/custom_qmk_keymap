# Apollo87H-delta LED indicator

## Features

- White LED works as indicator in caps/scroll lock
- Any other LED is turned off

## Troubleshooting

### If you fail to enter bootloader mode

You need to clear EEPROM by doing below.

    Plug out USB cable from the keyboard.
    Plug in the cable while holding down the ESC(or the top left) key.
    Plug out the cable again and then plug in.
